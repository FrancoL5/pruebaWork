Para usar la app se debe abrir una consola de comandos.
Puede ser el cmd o bash.
CMD: apretar tecla de windows + r y navegar hasta la carpeta con el archivo app dentro, para ver los comandos escribir:
        node app.js help
Bash: en la misma carpeta que está este readme, apretar click derecho e ir a la opcion de git BASH here, eso va a abrir una consola
    	apartir de ahi es lo mismo que arriba: node app.js help
